# FPV Drone Components Guide

A comprehensive, informative website containing detailed information about FPV drone components. This educational resource helps enthusiasts and newcomers understand the various components that make up an FPV drone system.

## Features

- **Component Categories**: Flight Controllers, ESCs, Motors, Frames, FPV Cameras, and Video Transmitters
- **Detailed Information**: In-depth explanations of each component type with technical specifications
- **Real Examples**: Includes specific component examples like the MatekSys F722-SE Flight Controller
- **Selection Guides**: Tables and guides to help choose the right components
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth transitions and intuitive user interface

## Components Covered

### Flight Controllers
- What they are and how they work
- Key features and specifications
- Featured example: MatekSys F722-SE with full specifications
- Selection considerations

### ESCs (Electronic Speed Controllers)

- Selection tips

### Motors

### Frames

- Low-latency camera technology

### Video Transmitters (VTX)

- Selection tips

## How to Use

Simply open `index.html` in a web browser to access the website. Navigate through different component categories by clicking on the component cards or using the navigation menu.

## Technology

- **HTML5**: Semantic markup for content structure
- **CSS3**: Modern styling with responsive design, flexbox, and grid layouts
- **JavaScript**: Interactive navigation and smooth user experience
- **No Dependencies**: Pure vanilla JavaScript, no external libraries required

## Purpose

This website is designed as an **informative educational resource** rather than a commercial shopping site. The goal is to provide comprehensive, unbiased information to help users understand FPV drone components and make informed decisions.
